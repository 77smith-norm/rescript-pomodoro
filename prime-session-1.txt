### Phase 1: Pure Logic (make the tests pass)

**1a. Create `src/Timer.res`** — pure state machine, no React.

The timer type (exact field names required by tests):
```res
type phase = Idle | Working | OnBreak | Paused

type timer = {
  phase: phase,
  timeLeft: int,
  workSecs: int,
  breakSecs: int,
  sessionCount: int,
}
```

Required functions (exact signatures):
```
Timer.makeInitial(~workSecs: int, ~breakSecs: int) -> timer
Timer.start(timer) -> timer          // Idle -> Working
Timer.pause(timer) -> timer          // Working/OnBreak -> Paused; Idle/Paused -> no-op
Timer.resume(timer) -> timer         // Paused -> Working
Timer.reset(timer) -> timer          // any -> Idle, timeLeft=workSecs, preserves sessionCount
Timer.tick(timer) -> (timer, bool)   // (newState, didComplete)
Timer.formatTime(seconds: int) -> string  // e.g. 90 -> "1:30", 61 -> "1:01", 0 -> "0:00"
```

Tick transition table (CRITICAL — implement exactly):
| Phase | timeLeft | After tick |
|-------|----------|------------|
| Working | > 1 | Working, timeLeft-1, completed=false |
| Working | 1 | OnBreak, timeLeft=breakSecs, sessionCount+1, completed=true |
| OnBreak | > 1 | OnBreak, timeLeft-1, completed=false |
| OnBreak | 1 | Working, timeLeft=workSecs, completed=true |
| Idle | any | Idle, unchanged, completed=false |
| Paused | any | Paused, unchanged, completed=false |

### Phase 2: App State + Reducer

Replace `src/App.res` with the full application. The state type:

```res
type state = {
  timer: Timer.timer,
  tasks: array<Tasks.task>,
  nextTaskId: int,
  newTaskTitle: string,
  settingsOpen: bool,
}

type action =
  | TimerStart
  | TimerPause
  | TimerResume
  | TimerReset
  | TimerTick
  | TaskAdd
  | TaskToggle(int)
  | TaskDelete(int)
  | SetNewTaskTitle(string)
  | OpenSettings
  | CloseSettings
  | UpdateWorkSecs(int)
  | UpdateBreakSecs(int)
```

Reducer correctness table (CRITICAL):
| Action | Effect |
|--------|--------|
| TimerStart | timer = Timer.start(timer) |
| TimerPause | timer = Timer.pause(timer) |
| TimerResume | timer = Timer.resume(timer) |
| TimerReset | timer = Timer.reset(timer) |
| TimerTick | let (t2, _) = Timer.tick(timer); timer = t2 |
| TaskAdd | if newTaskTitle != "": tasks = Tasks.addTask(tasks, newTaskTitle, nextTaskId); nextTaskId+1; newTaskTitle="" |
| TaskToggle(id) | tasks = Tasks.toggleTask(tasks, id) |
| TaskDelete(id) | tasks = Tasks.deleteTask(tasks, id) |
| SetNewTaskTitle(s) | newTaskTitle = s |
| OpenSettings | settingsOpen = true |
| CloseSettings | settingsOpen = false |
| UpdateWorkSecs(s) | timer.workSecs = s; if Idle: timer.timeLeft = s |
| UpdateBreakSecs(s) | timer.breakSecs = s |

Use `React.useReducer` for state. Use `React.useEffect1` for the interval ticker:
- Start interval when phase = Working
- Clear interval when phase ≠ Working or Paused
- Dispatch `TimerTick` every 1000ms
